<h1>Week 1 Assignment: Basic Android App</h1>

<h2>Project Overview</h2>
This basic Android app built using Jetpack Compose showcases a simple movie-related UI. The app allows users to view a film page, toggle between light and dark themes, and interact with UI elements such as play, download, and watchlist buttons.<br>
<br>
Additionally, the project includes Kotlin fundamentals, demonstrating object-oriented programming (OOP) concepts like classes, inheritance, interfaces, and collections.<br>

<h2>Setup Instructions</h2>

<h3>Prerequisites</h3>
- Android Studio (latest version recommended)<br>
- Kotlin installed in Android Studio<br>
- An emulator or a physical Android device for testing<br>

<h3>Steps to Run the Project</h3>
1- Clone the Repository<br>
git clone <repository-url><br>
cd Week1AssignmentBasicAndroidApp<br>
2- Open the project in Android Studio<br>
3- Build and Run the project using an emulator or physical device.<br>

<h2>Features Implemented</h2>
<h3>1- Android App Features:</h3>
- Movie Display Page: Shows movie details like title, year, resolution, and description.<br>
- Dark Mode / Light Mode Toggle: Switch themes dynamically using Material 3.<br>
- Play & Download Buttons: Simulates interaction for watching/downloading the movie.<br>
- Watchlist Toggle: Add/remove the movie from the watchlist.<br>
- Bottom Navigation Bar: Contains Browse, Watch, Downloads, and Search options.<br>
